# Cloudflare Workers 瘦身改造方案

## 问题

`npx wrangler deploy` 报错：
```
Your Worker exceeded the size limit of 3 MiB.
```

当前大小：**12.34 MiB**，免费限制：**3 MiB**

## 罪魁祸首

| 依赖 | 大小 | 说明 |
|------|------|------|
| `pydantic_core` | **4.3 MiB** | Pydantic 的 WebAssembly 核心（最大元凶） |
| `fastapi` + `starlette` | ~0.5 MiB | 引入了 pydantic |
| 其他（jose, passlib 等） | 较小 | 保留 |

**根因**：虽然 schemas 已经改成了 `dataclass`，但代码**仍然使用 `from fastapi import FastAPI`**，导致 FastAPI → Starlette → Pydantic → pydantic_core 整个依赖链被打包进去。

## 改造方案

**核心思路**：移除 FastAPI 和所有依赖它的代码，用纯 Python 标准库 + ASGI 手写一个极简框架。

### 1. 新建文件 `src/app/asgi_app.py`
一个 200 行左右的轻量 ASGI 框架，包含：
- `Router` 类（替代 FastAPI 的 `APIRouter`）
- `App` 类（替代 FastAPI 的 `FastAPI`）
- `HTTPException`、`StreamingResponse`
- CORS 支持
- JSON 请求体解析
- 路由匹配（支持路径参数 `{id}`）

### 2. 改造所有 router 文件（9个文件）

**改之前**（以 conferences.py 为例）：
```python
from fastapi import APIRouter, Depends, HTTPException
router = APIRouter(prefix="/conferences", tags=["conferences"])

@router.post("/")
async def create_conference(
    conference: ConferenceCreate,
    current_user: dict = Depends(get_current_active_user),
    db: D1Database = Depends(get_db),
):
    user_id = current_user["id"]
    ...
```

**改之后**：
```python
from app.asgi_app import Router, HTTPException
router = Router(prefix="/conferences")

@router.post("/")
async def create_conference(req, conference_id: str = ""):
    db = req.db
    user_id = req.user["id"]
    data = req.json(ConferenceCreate)
    ...
```

- `req.db` → D1 数据库实例
- `req.user` → 当前登录用户字典
- `req.json(DataclassType)` → 解析请求体为 dataclass
- `req.query_param("name")` → 获取查询参数
- `req.authorization` → Bearer token

### 3. 改造 `src/app/core/security.py`
移除 `Depends`、`OAuth2PasswordBearer` 等 FastAPI 依赖
→ 改为直接提供 `get_current_user(req)` 函数

### 4. 改造 `src/app/core/database.py`
移除 `from fastapi import Request`
→ 改为 `set_d1_binding(env)` + 直接获取 db

### 5. 更新 `pyproject.toml`
移除：`fastapi`、`starlette`、`pydantic`、`pydantic_core`
保留：`python-jose`、`passlib`、`python-dateutil`、`python-multipart`

### 6. 更新 `src/worker.py`
从 `from fastapi import FastAPI` 改为使用我们的 `App`

## 预估效果
